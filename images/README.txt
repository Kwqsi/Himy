HEIC from iPhone won't show in browsers. Convert HEIC → JPG first:

• Easiest (Mac): Open each HEIC in Preview → File → Export → Format: JPEG → Save.
• Or: Select all HEICs in Finder → Right‑click → Quick Actions → Convert to JPEG
  (if you have that option). Then rename the new JPEGs to photo5.jpg, photo6.jpg, etc.
• Or use a free site like cloudconvert.com (upload HEIC, download JPG).

After converting, put the JPGs in this folder as photo5.jpg, photo6.jpg, … photo13.jpg.
The website will show all of them in the scrolling strips.
